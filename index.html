CSS - Cascading Style Sheets

Syntax Selectors Properties Conclusion

CSS is a stylesheet language used to describe the presentation of a document written in HTML. It helps to design beautiful web pages using colors, fonts, spacing, layouts and animations.

CSS Syntax

CSS rules are written using selectors and declaration blocks. 

selector{
    property:value;
}

Example CSS

body{
    background-color:lightblue;
}

p{
    color:darkblue;
    font-size:18px;
}

  • External CSS

  • Class Selector

CSS Properties

  • color

  • font-size
  • margin
  • padding
  • border

Conclusion

CSS is very important for creating attractive and responsive websites. It improves user experience and makes websites look professional.
